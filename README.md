Created with create react app, to start run `npm install` and then `npm start`

Data is pulled from get requests to github api for each framework.


Users can vote for their favorite once per email


No styling was considered at all, just used bootstrap for a quick modal and basic styling so its not too ugly


You can click the button on the table for votes to see who voted for that framework (added button to be more noticeable)
